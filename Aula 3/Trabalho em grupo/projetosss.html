Glossário

1. Conceitos Gerais

1. HTML

HTML é uma linguagem de marcação que usa tags para rotular os elementos usados. Visa o conteúdo apresentado em um site, como os textos, parágrafos, imagens, etc.

2. Elemento

O elemento do HTML é a junção do conteudo (o que quero mostrar visualmente no site) com as tags (ferramentas que vão especificar como esse conteúdo vai ser apresentado).

3. Tag

Tags são marcações inseridas antes e depois do conteúdo apresentado no site, que vão formatar esse conteúdo para que fique visualmente adequado. Devem sempre estar entre "< >"

Exemplo:
<p>

2. Tags de definição

1. <p>
Cria um parágrafo, uma linha de texto
Exemplo: <p>Hello World!</p> Saída:

Hello World

2. <h1> -- <h6>
Define títulos e subtitulos de acordo com a numeração após a letra "h". Quanto menor o número, maior a letra.
Exemplo: <h1>H1</h1><h2>H2</h2><h3>H3</h3><h4>H4</h4><h5>H5</h5><h6>H6</h6>
Saída:

H1

H2

H3

H4

H5
H6
3. <blockquote>
Cria uma citação, com um recuo do texto original
Exemplo: <blockquote>Lorem ipsum dolor sit amet</blockquote> Saída:
Lorem ipsum dolor sit amet

3. Tags de formatação

1. <strong>

Deixa o conteúdo do elemento em negrito.

Exemplo:
<strong>Germinare</strong>

Saída:
Germinare

2. <ins>

Aplica um sublinhado no conteúdo do elemento.

Exemplo:
<ins>Germinare</ins>

Saída:
Germinare

3. <em>

Transforma o texto do conteúdo em itálico, normalmente usado para formatar palavras estrangeiras.

Exemplo:
<em>Germinare</em>

Saída:
Germinare

4. <code>

Formata o texto como um código usado em linguagem de programação

Exemplo:
<code>print("Germinare")</code>

Saída:
print("Germinare")

5. <del>

Risca uma palavra no meio, com o intuito de anulá-la no texto.

Exemplo:
<del>Germinare</del>

Saída:
Germinare

6. <q>

Deixa o conteúdo do elemento entre aspas. Usado para exemplificar citações.

Exemplo:
<q>Germinare</q>

Saída:
Germinare

7. <sup>

Torna o conteúdo em sobrescrito.

Exemplo:
Instituto<sup>Germinare</sup>

Saída:
InstitutoGerminare

8. <sub>

Torna o conteúdo em subscrito.

Exemplo:
Instituto<sub>Germinare</sub>

Saída:
InstitutoGerminare

4. Listas

1. <ol>

Cria uma lista ordenada de elementos.

2. <ul>

Cria uma lista de elementos sem ordem definida.

3. <dl>

Cria uma lista de definição. Para criar um item, utiliza-se a tag "dt" e para definí-lo, utiliza-se a tag "dd"

4. <li>

Define o conteúdo como um item da lista, seja ordenada ou não.

5. type=""

Define quais serão os tópicos dos itens da lista.
Em uma lista ordenada, pode ser "1", "I", "i", "a" e "A".
Em uma lista não ordenada, pode ser "circle" (círculo), "square" (quadrado), "disc" (ponto) e "none" (vazio).

6. Exemplo de lista ordenada

Exemplo:
<ol type="1">
<li>Instituto</li>
<li>Germinare</li>
</ol>

Saída:
  1. Instituto
  2. Germinare
7. Exemplo de lista não ordenada

Exemplo:
<ul type="circle">
<li>Instituto</li>
<li>Germinare</li>
</ul>

Saída:
  • Instituto
  • Germinare
8. Exemplo de lista de definição

Exemplo:
<dl>
<dt>Instituto Germinare</dt>
<dd>Uma escola inovadora<dd> <dl>

Saída:
Instituto Germinare
Uma escola inovadora

5. Outras tags

1. <pre>
Considera os espaços e tabs dados em um determinado espaço
Exemplo: 
<pre>Hello World!
        Hello Other World!!</pre>
Saída: 
Hello World
                      Hello Other World!!
2. <a>
Transforma o texto anexado em um link, podendo ser usado para:
  • Enviar para outra página externa
  • Enviar para uma parte da página atual
  • Enviar para outro arquivo html no repositório
Para indexar um link, utilizaremos a propriedade HREF e se abrirá uma nova guia ou sobrescreverá a atual utilizando TARGET
Exemplo: 
<a href="www.google.com" target = "_blank">Link externo</a><br>
<a href="#id do primeiro elemento" target = "_blank">Voltar para o começo</a>
Saída:
Link externo
Voltar para o começo
